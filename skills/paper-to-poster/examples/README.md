# Examples gallery (AXXX)

The four upstream posterly examples, **re-rendered in the AXXX brand** and used as
the skills-repo showcase. Each was produced by running the AXXX layer on the
original example:

```bash
python axxx/fetch_assets.py --dest examples/<name>/images   # affiliation logos + bullet
python axxx/apply_theme.py  examples/<name>/poster.html     # AXXX palette/accent bar/bullets/Inter
# then balanced until posterly's gates pass:
python tools/poster_check.py measure examples/<name>/poster.html
python tools/poster_check.py polish  examples/<name>/poster.html
python tools/render_preview.py       examples/<name>/poster.html --png examples/<name>/poster_preview.png
```

| Example | Venue / canvas | measure | polish |
|---|---|---|---|
| `powerflow_icml2026` | ICML 2026 · 60×36 landscape | PASS (spread 4.30 px) | 0 warnings |
| `tdgfn_icml2026` | ICML 2026 · 60×36 landscape | PASS (spread 4.80 px) | 1 accepted (FIG/TALL-SMALL, deliberate 2-panel figure) |
| `optail_neurips2024` | NeurIPS 2024 · 60×36 landscape | PASS (spread 0.89 px) | 0 warnings |
| `hello_world` | A2 portrait · reference | PASS (spread 0.00 px) | 0 warnings |

Each example dir holds `poster.html` (AXXX-themed, self-contained), `images/`
(real figures + the fetched-then-committed AXXX logos), `poster_preview.pdf/png`,
and `thumb.png`.

## Gallery site

[`index.html`](index.html) is the AXXX-styled landing page linking all four. The
repo-root workflow [`.github/workflows/pages.yml`](../../../.github/workflows/pages.yml)
publishes this directory to **https://axxx-institute.github.io/skills**.

## Regenerating

The four were regenerated by parallel subagents (one per example): theme + fetch
logos + swap the header affiliation strip to the AXXX set + rebalance to pass the
gates + render a thumbnail. To redo one, run the commands above and re-balance per
`SKILL.md` Step 4.

> Provenance: derivatives of posterly examples (AGPL-3.0). Author/venue content is
> kept from upstream; only the brand styling and affiliation logos are AXXX.
