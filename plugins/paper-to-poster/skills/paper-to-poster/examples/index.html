AXXX skills — Claude Code plugins

AXXX-Institute's Claude Code marketplace: two independently-installable plugins — paper-to-poster (AXXX-branded academic posters) and mlspace-jobs (MLSpace GPU job skills). Install only what you need.

Add the marketplace: /plugin marketplace add AXXX-Institute/skills

Plugins & skills

Each plugin ships one or more skills that Claude activates from their descriptions. The two plugins are independent — installing one does not pull in the other.

paper-to-poster AGPL-3.0

/plugin install paper-to-poster@axxx-institute
paper-to-poster POSTER
Turn a paper into an AXXX-branded, print-ready conference poster (single HTML/CSS → PDF), with affiliation logos fetched from this repo's asset releases. Passes posterly's measure + polish gates. See the gallery below.

mlspace-jobs MIT

/plugin install mlspace-jobs@axxx-institute
mlspace-jobs-scaffold BUILD
Scaffold experiments-as-code MLSpace training + eval launchers (run_train_jobs.py / run_eval.py / experiments.py) into a repo — an experiments-as-code registry, idempotent + in-progress-dedup submission, out-of-workdir artifacts, and code staging; verified by a green --dry run.
mlspace-jobs-quick-start SETUP
First-time MLSpace setup: an interactive, one-step-at-a-time walkthrough from conda env → mls install → credentials → submitting and monitoring a first job. Explicit-only — launch it with /mlspace-jobs-quick-start.
mlspace-jobs OPERATE
Day-to-day operation once mls is configured: a command reference for monitoring, logs, waiting, killing jobs, accelerate multi-GPU config, and troubleshooting.
